Instruction to create a bot with Botfather

Step 1:

In Telegram is Father Bot, which is main bot of bots and it will help us to reach a goal for creating our shop (https://core.telegram.org/bots#6-botfather). So, to create Bot, we must to open Telegram App and find by tag this bot – @BotFather

Step 2:

After entering a query in the search, will displayed list of results, where you must select the correct one. In our case it’s first. Please, be careful and don’t confuse with other variants! Since we did that, we will be translated to the Bot. Then we have to click on “START” button to activate him.

Step 3:

How we can see this @BotFather is multiple, so on answer we have list of commands with which help you can create your own bot for different needs.

Step 4:

In our case we are creating simple Bot, so to do that, we must to select in the list or write next command – “/newbot”.

By instruction of @BotFather, for first step we must to give name to our Bot. In this example I named it like “test-shop”. After we must to mention username of Bot. When we done it we will have link to the bot and API token which we will need to create shop. If you named your Bot wrong, just use commands witch are mentioned earliar.
